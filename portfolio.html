Project 1

The program accepts inputs of the number of scoops and flavors for the ice cream cone from the user. The program then uses turtle to draw the ice cream cone, adding a scoop of ice cream on top with a different color for each flavor. The scoops get smaller as more are added. Then, it adds a cherry on top.

Project 2

When the program is run, it shows the main menu, which allows you to select a game mode or see the instructions by clicking different buttons. The different game modes allow for 12, 18, and 24 cards. The first player gets to select 2 cards first, and if they match, then they are awarded a point. If they do not match, then Player 2 chooses 2 cards. This continues until all cards have been matched. The player with more points at the end wins."
